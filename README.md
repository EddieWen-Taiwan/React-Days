###Day-6 Slide
簡單的滑動 Slide~<br/>
這用 React 寫蠻不錯的，不用考慮現在是不是第一個 or 最後一個再去進行複製，每一次 state 改變時，DOM 只需要寫入現在顯示的 & 前一個 & 後一個就好。而這個設定在一開始寫 component 時就先寫好後面在改變 state 時都很輕鬆了。<br/>
<http://github.eddiewen.me/React-Days/DAY-6-Slide/>
